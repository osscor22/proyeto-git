1. Configuración del Entorno

El primer paso para cualquier proyecto de automatización es preparar el ecosistema de desarrollo. Necesitamos tres componentes principales:

  • Java Development Kit (JDK): Base para compilar y ejecutar código Java. Se recomienda Java 11 o superior.
  • IDE: Herramientas como IntelliJ IDEA, Eclipse o VS Code para escribir y gestionar el código de pruebas.
  • Apache Maven: Herramienta de gestión de dependencias que permite declarar librerías externas en el archivo pom.xml.

Una vez instalados, configura las variables de entorno (JAVA_HOME, MAVEN_HOME) para que sean accesibles desde cualquier terminal del sistema.
