4. Step Definitions en Java

Cucumber vincula cada frase del archivo .feature a un método Java mediante las anotaciones @Dado, @Cuando y @Entonces. Esos métodos contienen el código Selenium que interactúa con el navegador.

Crea la clase LoginSteps.java en:
src/test/java/steps/

☕ LoginSteps.java 
import io.cucumber.java.es.*;

    @Entonces("el sistema muestra el panel de control")
